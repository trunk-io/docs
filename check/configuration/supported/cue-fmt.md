# cue-fmt

[**cue-fmt**](https://cuelang.org) is a linter for Cue.

You can enable the cue-fmt linter with:

```shell
trunk check enable cue-fmt
```

## Auto Enabling

cue-fmt will be auto-enabled if any *Cue* files are present.
