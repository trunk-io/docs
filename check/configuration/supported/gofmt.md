# gofmt

[**gofmt**](https://pkg.go.dev/cmd/gofmt) is a linter for Go.

You can enable the gofmt linter with:

```shell
trunk check enable gofmt
```

## Auto Enabling

gofmt will be auto-enabled if any *Go* files are present.
