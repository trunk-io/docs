# opa

[**opa**](https://www.openpolicyagent.org/docs/latest/cli/#opa-fmt) is a linter for Rego.

You can enable the opa linter with:

```shell
trunk check enable opa
```

## Auto Enabling

opa will never be auto-enabled. It must be enabled manually.
