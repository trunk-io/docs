# golines

[**golines**](https://pkg.go.dev/github.com/segmentio/golines) is a linter for Go.

You can enable the golines linter with:

```shell
trunk check enable golines
```

## Auto Enabling

golines will never be auto-enabled. It must be enabled manually.
