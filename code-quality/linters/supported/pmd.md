# pmd

[**pmd**](https://pmd.github.io/) is a linter for Apex and Java.

You can enable the pmd linter with:

```shell
trunk check enable pmd
```
![pmd example output](./pmd.gif)
## Auto Enabling

pmd will never be auto-enabled. It must be enabled manually.
