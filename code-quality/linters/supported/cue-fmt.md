# cue-fmt

[**cue-fmt**](https://cuelang.org) is a formatter for Cue.

You can enable the cue-fmt formatter with:

```shell
trunk check enable cue-fmt
```

## Auto Enabling

cue-fmt will be auto-enabled if any _Cue_ files are present.
