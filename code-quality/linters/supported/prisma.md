# prisma

[**prisma**](https://github.com/prisma/prisma#readme) is a linter for Prisma.

You can enable the prisma linter with:

```shell
trunk check enable prisma
```

## Auto Enabling

prisma will never be auto-enabled. It must be enabled manually.
