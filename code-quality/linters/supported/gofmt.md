# Gofmt

[**Gofmt**](https://github.com/rhysd/actionlint) is a formatter for Go.

You can enable the Gofmt formatter with:

```shell
trunk check enable gofmt
```

## Auto Enabling

Gofmt will be auto-enabled if any *Go* files are present.
