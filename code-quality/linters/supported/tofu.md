# tofu

[**tofu**](https://github.com/opentofu/opentofu) is a linter for Terraform.

You can enable the tofu linter with:

```shell
trunk check enable tofu
```

## Auto Enabling

tofu will never be auto-enabled. It must be enabled manually.
