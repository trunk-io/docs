# Oxipng

[**Oxipng**](https://github.com/shssoichiro/oxipng) is a formatter for PNG.

You can enable the Oxipng formatter with:

```shell
trunk check enable oxipng
```

## Auto Enabling

Oxipng will be auto-enabled if any _Png_ files are present.
