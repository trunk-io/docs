# gofumpt

[**gofumpt**](https://pkg.go.dev/mvdan.cc/gofumpt) is a linter for Go.

You can enable the gofumpt linter with:

```shell
trunk check enable gofumpt
```

## Auto Enabling

gofumpt will never be auto-enabled. It must be enabled manually.
