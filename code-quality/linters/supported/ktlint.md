# ktlint

[**ktlint**](https://github.com/pinterest/ktlint#readme) is a linter for Kotlin.

You can enable the ktlint linter with:

```shell
trunk check enable ktlint
```

## Auto Enabling

ktlint will be auto-enabled if any _Kotlin_ files are present.
