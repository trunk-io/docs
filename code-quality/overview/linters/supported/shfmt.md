# shfmt

[**shfmt**](https://github.com/mvdan/sh#readme) is a linter for Bash.

You can enable the shfmt linter with:

```shell
trunk check enable shfmt
```

## Auto Enabling

shfmt will be auto-enabled if any _Shell_ files are present.
