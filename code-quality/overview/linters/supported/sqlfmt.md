# sqlfmt

[**sqlfmt**](https://github.com/tconbeer/sqlfmt#readme) is a linter for SQL.

You can enable the sqlfmt linter with:

```shell
trunk check enable sqlfmt
```

## Auto Enabling

sqlfmt will never be auto-enabled. It must be enabled manually.
